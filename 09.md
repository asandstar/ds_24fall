字符串逆序
Description
给定字符串 S ，请逆序输出字符串的单词序列。
单词即是非空格字符的序列。S 中的单词与单词之间间隔一个空格。
返回一个字符串，其中单词顺序颠倒，单词之间用一个空格分隔。
Input
字符串 S(1 <= |s| <= 10^5)
其中|s|表示字符串s的长度。它由可打印的ASCII字符组成。
Output
字符串 S‘ 单词顺序颠倒的输出字符串。
Sample Input 1 
Bob loves John
Sample Output 1
John loves Bob
Sample Input 2 
Hello world 
Sample Output 2
world Hello
Sample Input 3 
A good example
Sample Output 3
example good A
